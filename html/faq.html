Perguntas Frequentes

  • Qual é meu e-mail institucional?

    • O e-mail institucional é aquele que o usuário recebeu ao ser matriculado na Fatec. Para que seja válido, deverá ser como no exemplo abaixo:

      usuario@fatec.sp.gov.br

  • Como eu encontro meu e-mail institucional?

    • Seu e-mail institucional pode ser encontrado no portal academico SIGA. Ou entre em contato com a secretaria academica da Fatec ZL.

      f111acad@fatec.sp.gov.br

  • Qual o significado dos status dos chamados?

    • AGUARDANDO APROVAÇÃO

      Indica que o chamado está em análise pela equipe de TI.

      ABERTO

      Indica que o chamado está aberto para execução da solicitação.

      AGENDADO

      Indica que o serviço solicitado está agendado para execução da equipe de TI.

      AGUARDANDO RESPOSTA

      Indica que o chamado já foi executado e aguarda a validação do solicitante.

      CONCLUÍDO

      Indica que o chamado já foi executado pela equipe de TI.

      REJEITADO

      Indica que o chamado foi rejeitado pela equipe de TI.

      CANCELADO

      Indica que o chamado foi cancelado pela equipe de TI.

  • Como identificar o local onde está o equipamento com problema?

    • Para que a equipe de TI posso localizar mais facilmente o equipamento, informe:

      Departameto, Sala, Andar, Bloco

  • Como identificar o nome do computador?

    • Para identificar o nome do computador siga os passos abaixo:

      1 - Clique no botão do Windows localizado no canto inferior esquerdo.

      2 - Digite o comando cmd para abrir o prompt de comando.

      3 - Na janela do prompt de comando (CMD) digite o comando -> hostname e pressione Enter.

      Será exibido o nome do computador.

  • O que devo colocar na descrição do problema?

    • Para facilitar a tratativa da solicitação é importante que o máximo de informações sejam colocadas no campo de descrição do problema.

      Algumas dicas:

      1 - Informe a data que identificou a falha.

      2 - Informe sempre o nome do computador.

      3 - Informe se a falha impossibilita o uso total ou parcial do equipamento.

      4 - Indique se trata-se de uma falha de hardware ou software.

      OBS.: Caso seja uma necessidade de instalação de software, será analisado se trata-se de uma instalação previamente autorizada.

  • O que seria o tipo do equipamento?

    • Caso seja uma falha de hardware, é importante indicar se trata-se de CPU, Monitor, Mouse, Techado, Projetor Multimidia.
